# Comparative Technostress Triggers in Mobile Banking: Evidence from User Reviews in Indonesia and Thailand

This repository contains the datasets for the research article: **"Comparative Technostress Triggers in Mobile Banking: Evidence from User Reviews in Indonesia and Thailand"**. 

This study identifies and compares technostress triggers among users of two market-leading mobile banking applications: **Livin' by Mandiri** (Indonesia) and **K PLUS** (Thailand), utilizing a text mining approach combining Aspect-Based Sentiment Analysis (ABSA) and Latent Dirichlet Allocation (LDA).
